# iCHEF Lunch Bot

requires Python 3

## Usage

```shell
./lunch.py [path of lunch.yaml]
```

### Lunch.yaml format

```yaml
orders:
  Richard:
    - 牛肉捲
    - 雞肉沙拉
  小冠:
    - 牛肉捲
    - 雞肉沙拉
    - 蛤蜊湯
  于庭:
    - 蛤蜊湯
price:
  牛肉捲: 89
  雞肉沙拉: 159
  蛤蜊湯: 45
```

### Expected Output

```
>>> Item Summary =======================================================================================================
牛肉捲: 2
蛤蜊湯: 2
雞肉沙拉: 2
>>> Customer Summary ===================================================================================================
牛肉捲: Richard, 小冠
蛤蜊湯: 小冠, 于庭
雞肉沙拉: Richard, 小冠
>>> Price ==============================================================================================================
Overall price: $586
Richard: $248
小冠: $293
于庭: $45
```
